Проекты 2004 года - Электронная торговля для государственных нужд (государственные закупки)
1. Наименование инвестиционного проекта: Приобретение оборудования и создание программно-технического комплекса системы электронной торговли для осуществления закупок продукции для федеральных государственных нужд.

2. Ключевые слова: Система электронной торговли, портал государственных закупок

3. Цели и задачи проекта:

Общими целями создания и использования ЕС ЭГЗ и СЭТГН являются:

  • Повышение эффективности госзакупок за счет:
    • снижения затрат на проведение самого процесса государственных закупок;
    • ускорения процедур проведения конкурсных торгов.
  • Эффективное расходование бюджетных средств за счет:
    • повышения прозрачности госзакупок;
    • снижения коррупции и иных злоупотреблений при госзакупках;
    • увеличения конкуренции за получение государственных заказов (снижение цены поставщиков);
    • возможностей более широкого выбора на основании показателя качество/цена.
  • Интеграция России в глобальную мировую систему электронной торговли за счет:
    • Использования правовой базы, соответствующей международной;
    • Использования совместимых аппаратных и программных средств и форматов данных;
    • Унификации процедур закупок.
  • Главной задачей инвестиционного проекта является:
    • Комплектация программно-технического комплекса системы электронной торговли для осуществления закупок продукции для федеральных государственных нужд (ПТК СЭТГН) оборудованием, системным и базовым, а также прикладным программным обеспечением
4. Годы реализации проекта:

Реализация проекта по созданию первой очереди системы электронной торговли для осуществления закупок продукции для федеральных государственных нужд (СЭТГН) - 2002-2004 годы;

Реализация инвестиционного проекта по приобретению оборудования и созданию программно-технического комплекса системы электронной торговли для осуществления закупок продукции для федеральных государственных нужд - 2004 год.

5. Организация-исполнитель: ООО "УСП Компьюлинк".

6. Краткая информация о сути выполняемого проекта:

Реализация инвестиционного проекта по приобретению оборудования и созданию программно-технического комплекса системы электронной торговли для осуществления закупок продукции для федеральных государственных нужд является очередным этапом в процессе создания первой очереди системы электронной торговли для осуществления закупок продукции для федеральных государственных нужд.

Система электронной торговли для осуществления закупок продукции для федеральных государственных нужд (далее - СЭТГН) предназначена для комплексного обеспечения системы закупок товаров (работ, услуг) для государственных нужд в соответствии с российскими и международными законодательными актами и нормативными документами посредством использования современных информационно-коммуникационных технологий и программно-аппаратных средств для электронной торговли.

Система электронной торговли для осуществления закупок продукции для федеральных государственных нужд позволит автоматизировать эти процессы во всех федеральных органах исполнительной власти, значительно уменьшить издержки государственных заказчиков при проведении конкурсов, сократить потери и злоупотребления. При этом для государственных заказчиков будут созданы типовые комплексы аппаратных и программных средств, необходимые для проведения электронной торговли при осуществлении закупок продукции для федеральных государственных нужд.

По итогам реализации в 2004 году инвестиционного проекта по приобретению оборудования и созданию программно-технического комплекса системы электронной торговли для осуществления закупок продукции для федеральных государственных нужд достигнуты следующие результаты:

  • Поставлено оборудование и программное обеспечение;
  • Выполнен монтаж и настройка оборудования ПТК СЭТГН;
  • Установлено и настроено системное и базовое программное обеспечение ПТК СЭТГН;
  • Установлено и настроено прикладное программное обеспечение ПТК СЭТГН;
  • Доработана (актуализирована) эксплуатационная документация ИТЦ и ИИВС Минэкономразвития России.
7. Социально-экономический эффект: Применение СЭТГН заметно повысит эффективность использования средств федерального бюджета и бюджетов субъектов Российской Федерации при осуществлении государственных закупок, а также создаст предпосылки для широкого использования информационно-коммуникационных технологий (ИКТ) в процессе взаимодействия органов государственной власти и хозяйствующих субъектов. Ожидается, что экономия бюджетных средств после внедрения системы составит примерно 15 процентов.
