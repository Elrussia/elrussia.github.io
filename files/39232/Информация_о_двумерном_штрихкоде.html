Проекты 2004 года - Архитектура
1. Наименование проекта: Разработка предложений по стандарту обмена информацией между системами электронного документооборота органов государственной власти на основе двумерного штрихкода.

2. Ключевые слова: Двумерный штрихкод, сканер, документ, регистрация документа, регистрационная карточка, резолюция, исполнитель, руководитель, межведомственный обмен, сканирование.

3. Цели и задачи проекта:

Целью проекта является разработка единого механизма автоматизированного обмена информацией между СЭДО органов государственной власти, который позволит:

  • Автоматизировать процедуру регистрации входящих и внутренних документов в системы электронного документооборота (СЭДО).
  • Повысить скорость ввода документов в СЭДО организацией получателем.
  • Качественно уменьшить количество ошибок ручного ввода данных и соответственно затраты на их выявление и исправление.
  • Повысить скорость идентификации документа в СЭДО по его бумажному оригиналу.
Задачи проекта:
  • Определить список передаваемых регистрационных реквизитов документа.
  • Выбрать двумерную символику для кодирования информации.
  • Предложить варианты местоположения символов штрих-кода.
  • Предложить подходы к кодированию информации.
  • Разработать макет подсистемы обмена информацией на основе двумерного штрихкода.
  • Предложить план развития и внедрения решения.

4. Годы реализации проекта: 2004

5.Организация-исполнитель: ЗАО "Диджитал Дизайн"

6. Краткая информация о сути выполняемого проекта:

Краткое описание предметной области: Система автоматизирует деятельность сотрудников органов государственной власти, связанную с регистрацией, обработкой документов, взаимодействия систем электронного документооборота.

Решаемые проблемы:

  1. автоматизация процесса обмена документами между системами электронного документооборота;
  2. автоматизация процедуры регистрации входящих и внутренних документов в системы электронного документооборота (СЭДО);
  3. повышение скорости ввода документов в СЭДО организациями получателями документов;
  4. качественное уменьшение количества ошибок ручного ввода данных и затрат на их выявление и исправление;
  5. повышение скорость идентификации документа в СЭДО по его бумажному оригиналу.

Существующие и разрабатываемые аналогичные решения (мировой/российский опыт): Автоматизированная поисковая система ускоренного ввода документов ЗАО "РИТ СЕРВИС"

Результаты проекта в целом:

  1. Определен список передаваемых регистрационных реквизитов документа.
  2. Выбрана двумерная символика для кодирования информации.
  3. Предложены варианты местоположения символов штрих-кода.
  4. Предложены подходы к кодированию информации.
  5. Разработан макет подсистемы обмена информацией на основе двумерного штрихкода.
  6. Предложен план развития и внедрения решения.
Результаты проекта по годам: Результат работ в 2004 году приведен в пункте "Результаты проекта в целом".

7. Основные потребители:

Министерство экономического развития и торговли Российской Федерации, Российский Фонд федерального имущества, Федеральная служба по финансовым рынкам, другие органы государственной власти и организации.

Пользователями системы являются:

  • Сотрудники организаций корреспондентов
  • Регистраторы департаментов отделов ФОИВ (регистрирующие входящие документы).

8. Социально-экономический эффект:

  1. Повышение эффективности работы Министерства
    • уменьшение трудозатрат на внесение документов в систему электронного документооборота;
    • уменьшение количества ошибок, допускаемых пользователями при занесении документов в систему электронного документооборота.
  2. Снижение стоимости работы с документами.
