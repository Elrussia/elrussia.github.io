Проекты 2004 года - Другие проекты
1. Наименование проекта: НИОКР "Разработка методических рекомендаций по оценке эффективности результатов реализации ИКТ проектов".

2. Ключевые слова: Проект в сфере информационно-коммуникационных технологий (ИКТ проект), мероприятия ФЦП "Электронная Россия (2002-2010 годы)", эффективность результатов реализации проектов, методика

3. Цели и задачи проекта:

Цели:

  • Разработка методики оценки эффективности результатов реализации ИКТ проектов, включая функционально-стоимостной анализ;
  • Построение модели оценки эффективности результатов реализации ИКТ проектов на основе разработанных показателей и методики оценки;
  • Разработка методических рекомендаций по оценке эффективности результатов реализации ИКТ проектов ФЦП "Электронная Россия (2002-2010 годы)"

Задачи:

  • Проведение анализа существующих методик оценки эффективности ИКТ проектов;
  • Проведение классификация ИКТ проектов;
  • Подготовка предложений по системе показателей для оценки результативности и эффективности всех типов ИКТ проектов, механизмы расчета и применения показателей;
  • Разработка методики оценки эффективности ИКТ проектов, включая адаптированный для сферы информационно-коммуникационных технологий функционально-стоимостной анализ;
  • Построение модели оценки эффективности результатов реализации ИКТ проектов;
  • Адаптация модели оценки эффективности результатов реализации ИКТ проектов для программных мероприятий ФЦП "Электронная Россия (2002-2010 годы)";
  • Разработка методических рекомендаций по применению модели для проведения оценки эффективности результатов реализации проектов ФЦП "Электронная Россия (2002-2010 годы)";
  • Проведение тестовой оценки эффективности проекта в рамках программных мероприятий ФЦП "Электронная Россия (2002-2010 годы)" на основе разработанной модели, подготовка отчета и внесение корректировок в модель оценки и методические рекомендации по ее применению

4. Годы реализации проекта: 2004 год

5. Организация-исполнитель: ЗАО "Диджитал Дизайн"

6. Краткая информация о сути выполняемого проекта:

6.1. Краткое описание предметной области: Управление реализацией ФЦП "Электронная Россия (2002-2010 годы)"

6.2. Решаемые проблемы: Поддержка принятия решений при отборе проектов для реализации, оценке их результатов, корректировке ориентиров реализации ФЦП

6.3. Существующие и разрабатываемые аналогичные решения (мировой/российский опыт);

При анализе мирового опыта было показано разнообразие используемых в мире подходов и методик оценки эффективности результатов ИКТ проектов; в то же время подробной информации о конкретных методиках проведения такой оценки при реализации масштабных программ по информатизации обнаружить не удалось

6.4. Результаты проекта в целом;

В результате проведенной НИОКР, рекомендована к применению в рамках управления реализацией ФЦП "Электронная Россия (2002-2010 годы)" методика оценки эффективности результатов реализации проектов Программы, протестированная на ряде проектов, завершенных в 2003 году

7. Основные потребители: Лица, принимающие решения по реализации мероприятий в рамках программ по информатизации (в частности, ФЦП "Электронная Россия (2002-2010 годы)")

8. Социально-экономический эффект (ожидаемый - в случае практического применения результатов НИОКР):

Предоставление объективной информации о прогнозируемой / достигнутой эффективности ИКТ проектов, реализуемых в рамках ФЦП "Электронная Россия (2002-2010 годы)", необходимой для обоснованного принятия решений по отбору наиболее приоритетных тем для проектов, реализуемых в рамках Программы, по оценке их результатов и корректировке направлений дальнейшего развития мероприятий Программы.
